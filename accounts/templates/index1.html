The project is made for Natural Language Processing. It includes Twitter news extraction through tweets, Tweets Processing (cleaning), Sentiment Analysis, Language detection, and Language Translation. UI (User Interface) is made using HTML, CSS and JavaScript, and the functionality is provided using a very popular python framework called Django.

a) The main objective of this project is to display the new latest trends.

b) To determine the opinion of mass in positive or negative towards the subject of interest.

c) Graphical representation of the analysis in the form of bar graph, pie-chart.
